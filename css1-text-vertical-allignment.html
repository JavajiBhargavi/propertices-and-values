Vertical Alignment

The vertical-align property sets the vertical alignment of an element.

Example

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<style>
img.a {
vertical-align: baseline;
}
img.b {
vertical-align: text-top;
}
img.c {
vertical-align: text-bottom;
}
img.d {
vertical-align: sub;
}
img.e {
vertical-align: super;
}
</style>
<body>
<h3>vertical-align: baseline (default) <:/h3>
<p>The<img class="a" src="file:///C:/Users/key/Pictures/images%20(1).png" width="9" height="9">fox and The Grapes</p>
<h3>vertical-align: text-top:</h3>
<p>A<img class="b" src="file:///C:/Users/key/Pictures/images%20(1).png" width="9" height="9"> fox saw ripe grapes.</p>
<h3>vertical-align: text-bottom:</h3>
<p>He<img class="c" src="file:///C:/Users/key/Pictures/images%20(1).png" width="9" height="9">tried to jump to reach them but could not.</p>
<h3>vertical-align: sub:</h3>
<p>When<img class="d" src="file:///C:/Users/key/Pictures/images%20(1).png" width="9" height="9"> we can not get something we want, we pretend we don't like it.</p>
<h3>vertical-align: sup:</h3>
<p>A<img class="e" src="file:///C:/Users/key/Pictures/images%20(1).png" width="9" height="9"> fox saw ripe grapes.</p>
</body>
</html>



vertical-align: baseline (default):

The fox and The Grapes

vertical-align: text-top:

A fox saw ripe grapes.

vertical-align: text-bottom:

He tried to jump to reach them but could not.

vertical-align: sub:

When we can not get something we want, we pretend we don't like it.

vertical-align: sup:

A fox saw ripe grapes.
