HTML Lists

Lists are 2 types

1.Ordered List

2.Unordered List

Ordered List Start with the < ol >tag.Each list item start with the < li >tag.The list items will be marked with bullets .

Unordered List Start with the < ul >tag.Each list item start with the < li >tag.The list items will be marked with numbers .

Example

Unordered list

  • Flowers
  • Fruits
  • Trees

Ordered list

  1. Flowers
  2. Fruits
  3. Trees
