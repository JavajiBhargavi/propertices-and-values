HTML Headings

HTML headings are defined with the < h1 > to < h6 > tags.

h1,h2,h3,h4,h5,h6

Example

< html >
< head >
< title >< /title >
< /head >
< body >

< h1 >Tajmahal< /h1 >

< h2 >Qutbminar< /h2 >

< h3 >Charminar< /h3>

< h4 >Golkonda< /h4 >

< h5 >Red fort< /h5 >
< h6 >India gate< /h6>
< /body > < /html >
