Specify colors using RGB values

rgb(255, 0, 0) red

rgb(0, 0, 255) blue

Shades of gray

By using equal values for red, green, and blue, you will get different shades of gray:

rgb(60, 60, 60)

rgb(90, 90, 90)

rgb(120, 120, 120)

rgb(180, 180, 180)

rgb(210, 210, 210)

rgb(240, 240, 240)

Make transparent colors with RGBA

rgba(100, 220, 60, 0)

rgba(100, 220, 60, 0.1)

rgba(100, 220, 60, 0.3)

rgba(100, 220, 60, 0.5)

rgba(100, 220, 60, 0.7)

rgba(100, 220, 60, 0.9)
