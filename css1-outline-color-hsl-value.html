HSL value

The outline color can also be specified using a HSL value.

Example

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<style>
p.ex1 {
border: 1px solid black;
outline-style: solid;
outline-color:hsl(0, 100%, 50%);
}
p.ex2 {
border: 1px solid black;
outline-style: dotted;
outline-color: hsl(0, 59%, 41%) ;
}
p.ex3 {
border: 1px solid black;
outline-style: outset;
outline-color:hsl(0, 0%, 73%);
}
</style>
<body>
<h1>The golden egg</h1>
<p class="ex1">A farmer once had a magical goose.</p >
<p class="ex2">Every day that goose would lay one golden egg. </p>
<p class="ex3">He used to sell that golden egg in the market. </p>
</body>
</html>
